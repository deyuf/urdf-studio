# Quickstart

Pick the entry that matches how you want to use URDF Studio.

- [Browser](./web.html) — open the deployed URL, click *Open Folder*, pick a
  ROS package. Nothing is uploaded; the page reads files locally.
- [VS Code](./vscode.html) — install from the Marketplace, open any
  `.urdf` / `.xacro` file. The custom editor handles the rest.

The rest of this section walks through each target individually.
