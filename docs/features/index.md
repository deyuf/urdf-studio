# Features

URDF Studio's features fall into five groups. Each has its own page below
with controls, behavior, and edge cases described in detail.

## Viewing the robot

- [Viewport & navigation](./viewport.html) — orbit, pan, zoom, preset
  cameras, fit, up-axis, wireframe, grid, axes.
- [Render modes](./render-modes.html) — visual vs. collision geometry.
- [Frames & inertia overlays](./overlays.html) — per-link TF axes, inertia
  ellipsoids, total CoM.

## Interacting

- [Joint controls](./joints.html) — sliders, numeric inputs, limit modes,
  search/filter, mimic propagation.
- [Inspector & link tree](./inspector.html) — selection, link details,
  parent/child relationships, mesh paths.

## Authoring & analysis

- [Diagnostics (Checks panel)](./diagnostics.html) — error catalog, what
  triggers each one, how to silence them.
- [Tools](./tools.html) — reachability sampling, never-colliding pair
  detection, SRDF authoring.

## Persisting work

- [Pose, bookmarks, export](./poses.html) — save pose, named bookmarks,
  JSON export, PNG screenshots, semantic states from SRDF.

## Configuration

- [Settings](./settings.html) — render mode default, up axis, xacro args,
  package roots, semantic files.
